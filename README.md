# 📱 Mobile Store Management System

A complete **Mobile Store Management System** built using **Laravel**, **PHP**, **MySQL**, **Bootstrap**, **HTML**, **CSS**, and **JavaScript**. This project provides a modern e-commerce platform where customers can browse products, place orders, and leave reviews, while administrators can efficiently manage the entire store through a dedicated admin dashboard.

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/dabhijinal/mobile_store.git
```

2. Move into the project directory

```bash
cd mobile_store
```

3. Install dependencies

```bash
composer install
```

4. Copy the environment file

```bash
cp .env.example .env
```

5. Generate application key

```bash
php artisan key:generate
```

6. Configure the database in the `.env` file.

7. Run migrations

```bash
php artisan migrate
```

8. Start the development server

```bash
php artisan serve
```
